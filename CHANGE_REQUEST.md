## 중복·충돌·모호성
- 변경 요청의 실제 문장이 비어 있어 모든 항목이 `[사람 확인 필요]` 상태다.
- 마지막 정상 배포 commit·URL은 현재 확인되지 않아 배포 기준점이 모호하다.
- 첨부 자료가 없어서 재현 방법과 디자인 판단이 아직 모호하다.

## 정렬 기준
- 1순위: 실제 변경 요청 문장 확정
- 2순위: 재현 자료와 첨부 파일 확인
- 3순위: 마지막 정상 배포 commit·URL 확인
- 4순위: 배포본과 로컬본 차이 정리
